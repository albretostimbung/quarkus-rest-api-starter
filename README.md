# Quarkus REST API Starter

Proyek starter untuk membangun REST API menggunakan Quarkus.

## Teknologi
- Java 21 (LTS)
- Quarkus
- RESTEasy Reactive
- Hibernate ORM Panache
- PostgreSQL
- Maven

## Cara Menjalankan

Jalankan perintah berikut di terminal:

```bash
./mvnw quarkus:dev
```

Aplikasi akan berjalan di:

```
http://localhost:8080
```
